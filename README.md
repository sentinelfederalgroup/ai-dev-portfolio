# AI Developer Portfolio

This repo is the control center for building a serious AI developer portfolio.
It tracks the projects, proof points, demos, and next work needed to look like
someone who can ship useful AI systems rather than only run notebooks.

## Portfolio thesis

Build practical AI tools with real product surfaces:

- local-first AI agents that do useful work without paid API tokens
- streamer, creator, and operator tools with polished interfaces
- retrieval, automation, moderation, and workflow agents
- clear architecture docs, tests, screenshots, and deployable demos

## Target portfolio shape

The goal is 6-8 repos that each show a different high-value AI engineering skill.

1. **Streamer Agent**: real-time assistant/moderator for Twitch/Kick/YouTube.
2. **Local RAG Desk**: local document search and answer app using embeddings and citations.
3. **Agent Workflow Runner**: task planner that calls tools, keeps state, and writes audit logs.
4. **AI Evaluation Harness**: prompt/model eval runner with golden tests and regression reports.
5. **Support Inbox Agent**: triages emails/tickets into actions, drafts replies, and tracks confidence.
6. **Computer Vision Utility**: image classifier or screenshot QA tool with a small UI.
7. **Open Source Contribution**: meaningful PR to an existing active repo.
8. **Portfolio Site**: live site tying everything together with demos, screenshots, and writeups.

## Quality bar

Every serious repo should have:

- README with a clear problem statement and demo path
- screenshots or GIFs
- architecture notes
- tests for core logic
- no fake claims about unsupported capabilities
- issue list or roadmap
- simple local run instructions
